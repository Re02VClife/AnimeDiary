# 图片管理需求

> 文件: features/image-management/

## 功能

- **上传**: 本地图片保存到 `images/{番剧名}/` 目录
- **删除**: 从本地目录移除图片文件
- **设为海报**: 将图片设置为番剧封面海报
- **高清预览**: 支持双击查看原图大图
- **命名规则**: `{番剧名}_{编号}.{ext}`

## API 端点

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/images/list` | POST | 列出某番剧的所有图片 |
| `/api/images/save` | POST | 保存图片到本地 |
| `/api/images/delete` | POST | 删除指定图片 |
